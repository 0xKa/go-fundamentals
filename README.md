# Go Fundamentals

A growing collection of small Go practices with matching Markdown notes.

## Run the Practice CLI Tool

Run the CLI from the repository root and enter a practice number:

```powershell
go run .
```

Alternatively, provide the number directly:

```powershell
go run . 3
```
